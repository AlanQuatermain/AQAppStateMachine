AQNotifyingBitfield Class Reference

Inherits from AQBitfield : NSObject
Declared in AQNotifyingBitfield.h

Overview

A bitfield which supports calling notification callback blocks whenever bits within certain ranges are modified.

Tasks

  • – notifyModificationOfBitsInRange:usingBlock:

    Install a notifier block for a given range of a bitfield.
  • – removeNotifierForBitsInRange:

    Remove a notifier for a specific range.
  • – removeAllNotifiersWithinRange:
