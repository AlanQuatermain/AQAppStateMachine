initWithRanges:masks:matchingValues:

Initialize a new descriptor.

- (id)initWithRanges:(NSArray *)ranges masks:(NSArray *)masks matchingValues:(NSArray *)values

Parameters

ranges

An array of AQRange objects specifying ranges to match.

masks

An array of masks to apply when matching the ranges.

values

An array of AQBitfield objects providing masks to apply to the ranges. May be nil.

Return Value

The newly-initialized instance.

Discussion

If masks are only required for some but not all entries in the ranges array, the corresponding entry in the masks array should be an NSNull instance.
