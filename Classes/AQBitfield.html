Properties

count

The number of significant bits within the bitfield.

@property (nonatomic, readonly) NSUInteger count

Discussion

NB: This returns the number of bits in the range which covers all 1 bits in the bitfield. i.e. it will return ((index of highest 1-bit) + 1).
